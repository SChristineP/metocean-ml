# metocean-ml 🌊🤖 
Machine Learning Toolkit for Metocean Data

metocean-ml is a powerful, open-source tool designed to apply Machine Learning (ML) to meteorological 🌦️ and oceanographic 🌊 data. It specializes in efficient (low-cost) downscaling—enhancing the spatial and temporal resolution of metocean datasets—bridging the gap between coarse global/regional models and high-resolution coastal applications.
